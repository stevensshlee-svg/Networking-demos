# Overview
This repository contains networking demonstrations that reflect concepts I learned while studying for the CCNA. The goal of these labs are to reinforce foundational networking knowledge through hands-on configuration and troubleshooting.

The scenarios focus on core networking topics commonly encountered in entry level networking and IT support roles, including switching, routing, VLAN configuration, DHCP, DNS, and basic network troubleshooting.

Each lab is built and tested in Cisco Packet Tracer and includes documentation of the configuration process, troubleshooting steps, and key takeaways. These demos are intended to show both my understanding of networking fundamentals and my ability to apply them in practical scenarios.
